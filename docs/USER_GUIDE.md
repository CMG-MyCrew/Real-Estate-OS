# REOS Enterprise User Guide

## Overview

REOS Enterprise is a real estate operating system for agents, teams, brokerages, investors, rental managers, admins, clients, and vendors.

Most work starts from the **REOS** menu in Google Sheets.

## Main Apps

### Dashboard

Use the dashboard for executive KPIs and operating status.

### Agent Portal

The Agent Portal is the daily command center for:

- Leads
- Tasks
- Transactions
- Commissions
- Rentals
- Documents
- Quick actions

### CRM App

Use CRM to manage contacts, leads, activities, follow-ups, and pipeline.

### Tasks App

Use Tasks to manage overdue, due today, and upcoming work.

### Transactions App

Use Transactions to track active deals, closing dates, commission, documents, and risk.

### Investments App

Use Investments to evaluate acquisition opportunities and property strategies.

### Rentals App

Use Rentals to manage rental properties, leases, maintenance, occupancy, and cash flow.

### Finance App

Use Finance to track income, expenses, commissions, taxes, and cash flow.

### Documents App

Use Documents to manage document records, Drive URLs, eSignature status, and missing document checklists.

### AI Assistant

Use the AI Assistant for:

- Daily briefings
- Lead coaching
- Transaction risk reviews
- Investment reviews
- Rental portfolio reviews
- Finance reviews
- Document summaries
- Email drafts

### Mobile App Preview

The mobile shell provides a phone-friendly interface for home, tasks, CRM, deals, and AI.

## Portals

### Client Portal

Clients can access controlled record data, documents, status updates, and portal messages.

### Vendor Portal

Vendors can view assignments, update status, add invoice URLs, and submit completion notes.

### Agent Portal

Internal team workspace for agents and staff.

## Daily Workflow

1. Open **Agent Portal**.
2. Review KPIs.
3. Clear overdue tasks.
4. Review due today tasks.
5. Update lead pipeline.
6. Check active transactions.
7. Review documents and signatures.
8. Use AI Daily Briefing for priorities.

## Lead Workflow

1. Add or import contact.
2. Create lead.
3. Set source, priority, and status.
4. Start follow-up sequence.
5. Track activities.
6. Convert to transaction when ready.

## Transaction Workflow

1. Create transaction record.
2. Add parties, address, closing date, and commission.
3. Add required documents.
4. Track tasks and deadlines.
5. Review transaction risk.
6. Close and record commission.

## Rental Workflow

1. Add rental property.
2. Track occupancy and rent.
3. Add lease records.
4. Track maintenance.
5. Review portfolio risk and cash flow.

## Document Workflow

1. Create document record.
2. Attach Drive URL.
3. Mark verification status.
4. Track signature status.
5. Use missing document reports.

## AI Usage Tips

AI responses use REOS context. Review outputs before sending messages or making decisions. Use AI for summaries, coaching, and prioritization, not as a replacement for professional judgment.

## Getting Help

Open **Help Center** from the REOS menu when available, or ask an administrator to review system diagnostics and permissions.
