# REOS Enterprise v3.2.4 — Client / Lender Portal UI

This sprint adds client and lender portal dashboards.

## Files

| File | Purpose |
| --- | --- |
| `src/ClientLenderPortal.gs` | Client/lender portal dashboard data, updates, and notes. |
| `src/ClientLenderPortal.html` | Admin UI for loading client/lender portal dashboards. |
| `docs/V3_2_4_CLIENT_LENDER_PORTAL_UI.md` | Documentation. |

## New Sheets

| Sheet | Purpose |
| --- | --- |
| `CLIENT_LENDER_PORTAL_UPDATES` | Client/lender-facing updates. |
| `LENDER_PORTAL_NOTES` | Lender notes and property references. |

## Features

- Client and lender dashboard data
- Property visibility
- Shared documents
- Messages
- Tasks
- Updates
- Lender notes

## Menu Access

`REOS → Open Client/Lender Portal`

## Acceptance Criteria

- Client/lender sheets are created automatically.
- Dashboard loads only for Client or Lender portal accounts.
- Properties, documents, messages, tasks, updates, and notes display.
